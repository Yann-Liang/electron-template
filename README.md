# electron-template

> electron-template
> description:
> electron-vue项目的模版，客户端的开发基于此模版开发。

#### Example
使用命令 `git checkout game-2048`切换到该分支，有一个基于此写的小游戏可供参考！

#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
